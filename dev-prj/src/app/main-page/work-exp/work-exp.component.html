Выполняемые мною работы в качестве фриланс разработчика

  • Верстка информационных сайтов, интернет-каталогов, landing-page по макетам и прототипам.
  • Наполнение сайтов контентом.
  • Сопровождение, внесение изменений и разработка дополнительных модулей на сайтах по требованию заказчика.

При разработке сайтов мною используются знания следующих технологий:

  • Язык гипертекстовой разметки HTML5 и каскадная таблица стилей CSS3.
  • Нарезка макетов средствами Adobe Photoshop, разработка проектов кодовым редактором Brackets и WebStorm, используя набор сниппетов Emmet.
  • Семантическая, валидная, адаптивная и отзывчивая верстка сайтов, а также табличная верстка.
  • Работа со спрайтами, CSS-анимации, CSS-фреймворк Bootstrap 3, система контроля версий Git при работе с Github, препроцессор SASS и инструмент сборки модулей webpack, CSS методология БЭМ и RSCSS.
  • Использование языка программирования Javascript, фреймворк для разработки single page application Angular 4, а также разработка модулей проектов при помощи технологии Ajax и библиотеки jQuery.
